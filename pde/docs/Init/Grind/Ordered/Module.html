    Prop

    Addition is compatible with a preorder if a ≤ b ↔ a + c ≤ b + c.

    • add_le_left_iff {a b : M} (c : M) : a ≤ b ↔ a + c ≤ b + c

      a + c ≤ b + c iff a ≤ b.
