    Difference of finite sets #

    Main declarations #

    • Finset.instSDiff: Defines the set difference s \ t for finsets s and t.
    • Finset.instGeneralizedBooleanAlgebra: Finsets almost have a boolean algebra structure

    Tags #

    finite sets, finset
