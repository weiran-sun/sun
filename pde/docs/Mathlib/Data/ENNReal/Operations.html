    Properties of addition, multiplication and subtraction on extended non-negative real numbers #

    In this file we prove elementary properties of algebraic operations on ℝ≥0∞, including addition, multiplication, natural powers and truncated subtraction, as well as how these interact with the order structure on ℝ≥0∞. Notably excluded from this list are inversion and division, the definitions and properties of which can be found in Mathlib/Data/ENNReal/Inv.lean.

    Note: the definitions of the operations included in this file can be found in Mathlib/Data/ENNReal/Basic.lean.
